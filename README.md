# 🏦 Banking Analysis Using Python

This project is a Banking Data Analysis project created using Python, MySQL, Pandas, Matplotlib, and Seaborn.

The main purpose of this project is to analyze banking customer data and generate useful business insights using data analytics and visualization techniques.

This project helps understand:
- Customer behavior
- Account usage
- Transaction trends
- Loan analysis
- Banking performance

---

# 📌 Project Objective

The objective of this project is to:

- Connect Python with MySQL database
- Analyze banking datasets
- Perform data cleaning and processing
- Generate business insights
- Create charts and visualizations

This project simulates real-world banking analytics used in financial institutions.
